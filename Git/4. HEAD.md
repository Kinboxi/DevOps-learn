**HEAD** — указатель на то место где ты сейчас находишься в Git
Обычно указывает на последний коммит активной ветки

---

**Detached HEAD** — когда ты на конкретном коммите, а не на ветке
Можешь смотреть старый код, редактировать, коммитить — но эти коммиты "висят в воздухе" пока не создашь ветку

---

## `git checkout [хеш]`
Переключиться на конкретный коммит
> пример: `git checkout 1a2b3c4`

Git предупредит:
```
You are in 'detached HEAD' state...
```

---

**Визуализация**
```
main:  A──B──C──D  ← обычная работа (HEAD на ветке)

main:  A──B──C──D
            ↑
          HEAD detached (переключился на C)
```

---

**Важно помнить**
- Работать можно, но коммиты висят в воздухе
- Если не создашь ветку — после переключения потеряешь изменения
- Это не ошибка — удобно чтобы посмотреть старую версию кода
