# Ветка(Branch) — это отдельная линия разработки, или отдельная версия проекта.
## Она помогает работать отдельно от главной версии, тоесть ты можешь создать новую ветку и там протестировать что то, независимо от главной версии, и если тебе понравилось, ты можешь соединить ветки. 

---- 

# `git branch "имя ветки"` — создает новую ветку но не переключается на нее
## пример : `git branch bugs`

---- 
# `git checkout "имя ветки"` — переключиться на указаную ветку.
## пример: `git checkout bugs`
