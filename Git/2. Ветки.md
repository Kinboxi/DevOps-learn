** Ветка (Branch) — отдельная линия разработки, независимая версия проекта
Создаёшь ветку → тестируешь там что угодно → если всё ок, сливаешь с главной **

---

### `git branch [имя_ветки]`
Создаёт новую ветку, но не переключается на неё

> 📌 `git branch bugs`

### `git checkout [имя_ветки]`
Переключиться на указанную ветку

> 📌 `git checkout bugs`
