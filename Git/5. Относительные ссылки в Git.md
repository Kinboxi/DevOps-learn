**Относительные ссылки** — способ перемещаться по истории без копирования длинных хешей
`^` означает один шаг назад к родителю

---

**Визуализация**
```
A──B──C──D (HEAD)
            ↑
  HEAD^  → C
  HEAD^^ → B
```

---

## `git log --oneline`
Показывает историю коммитов кратко — хеш и сообщение в одну строку

---

## `git checkout HEAD^`
Переход к предыдущему коммиту
> пример: `git checkout HEAD^^` — два шага назад

---

## `git show HEAD^`
Показывает что было изменено в предыдущем коммите

---

## `git diff HEAD^`
Сравнивает текущий коммит с предыдущим

---

**Зачем нужно**
Вчерашний коммит что-то сломал → быстро прыгаешь назад и смотришь как было до изменений — без копирования хешей
