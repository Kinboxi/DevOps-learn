# Перенаправление

# `>` — записать вывод в файл
# `>>` — дописать вывод в файл
# `<` — взять ввод из файла
# `|` — передать вывод одной команды в другую

---

## Примеры

**`ls > list.txt`** — сохранил список файлов в list.txt
**`echo "привет" > file.txt`** — записал текст в файл
**`echo "ещё" >> file.txt`** — дописал текст в файл

**`sort < names.txt`** — отсортировал из файла

**`ls | grep ".txt"`** — нашёл только txt файлы
**`cat file.txt | wc -l`** — посчитал строки в файле
**`cat /etc/passwd | grep root`** — нашёл root в passwd
