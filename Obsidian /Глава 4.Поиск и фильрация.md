# Поиск и фильтрация

# `grep` — ищет текст внутри файла
# `find` — ищет файлы и папки в системе
# `sort` — сортирует строки
# `uniq` — убирает дубликаты
# `wc` — считает строки, слова, символы
# `cut` — вырезает колонки из текста
# `head` — показывает первые строки файла
# `tail` — показывает последние строки файла
# `less` — просмотр файла постранично

---

## Дополнения

### grep
**`grep слово файл`**
**`grep -i слово файл`** — без учёта регистра
**`grep -r слово папка/`** — по всей папке
**`grep -n слово файл`** — с номерами строк

### find
**`find . -name "файл"`** — искать в текущей папке
**`find / -name "файл"`** — искать везде
**`find . -name "*.txt"`** — все файлы с расширением
**`find . -type f`** — только файлы
**`find . -type d`** — только папки

### sort
**`sort файл`** — по алфавиту
**`sort -r файл`** — в обратном порядке
**`sort -n файл`** — по числам

### uniq
**`uniq файл`** — убрать дубликаты
**`uniq -c файл`** — посчитать повторения
> ⚠️ работает только с отсортированным файлом — `sort файл | uniq`

### wc
**`wc файл`** — строки, слова, символы
**`wc -l файл`** — только строки
**`wc -w файл`** — только слова

### cut
**`cut -d: -f1 файл`** — первая колонка, разделитель `:`
**`cut -d, -f2 файл`** — вторая колонка, разделитель `,`

### head / tail
**`head файл`** — первые 10 строк
**`head -n 5 файл`** — первые 5 строк
**`tail файл`** — последние 10 строк
**`tail -n 5 файл`** — последние 5 строк
**`tail -f файл`** — следить за файлом живьём

### less
**`less файл`** — открыть файл
> стрелки — листать / `q` — выйти / `/слово` — поиск внутри

## Примеры

**`grep "error" log.txt`** → все строки с error
**`grep -r "root" /etc/`** → поиск по всей папке

**`find . -name "*.log"`** → все лог файлы
**`find / -name "passwd"`** → найти файл passwd

**`sort names.txt | uniq -c`** → отсортировал и посчитал повторения

**`wc -l file.txt`** → `42`

**`cut -d: -f1 /etc/passwd`** → только логины из файла passwd

**`head -n 5 file.txt`** → первые 5 строк
**`tail -f /var/log/syslog`** → живой просмотр лога

**`less /etc/passwd`** → открыл и листаешь
