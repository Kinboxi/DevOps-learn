# Пользователи и группы

# `whoami` — имя текущего пользователя
*`whoami` → `user`*

# `id` — UID, GID и все группы пользователя
*`id` → `uid=1000(user) gid=1000(user) groups=1000(user),27(sudo)`*

# `groups` — список групп пользователя
*`groups` → `user sudo docker`*

# `su` — сменить пользователя
*`su ivan` → переключился на Ивана*

# `sudo` — выполнить команду от имени root
*`sudo apt install vim` → установил пакет от root*

# `useradd` — создать пользователя
*`useradd -m ivan` → создал пользователя с папкой*

# `usermod` — изменить пользователя
*`usermod -aG programmer ivan` → добавил в группу sudo*
*`usermod -L ivan` → заблокировал*
*`usermod -U ivan` → разблокировал*

# `userdel` — удалить пользователя
*`userdel ivan` → удалил пользователя*
*`userdel -r ivan` → удалил с домашней папкой*

# `passwd` — сменить пароль
*`passwd ivan` → задал пароль Ивану*

# `groupadd` — создать группу
*`groupadd programmer` → создала группу devs*

# `groupdel` — удалить группу
*`groupdel programmer` → удалила группу devs*

# `visudo` — редактировать права sudo
*`visudo` → открыл файл sudoers*
