# Операции с файлами и каталогами

# `cp` — копирует файл или папку
# `mv` — перемещает или переименовывает файл
# `rm` — удаляет файл или папку
# `touch` — создаёт пустой файл
# `mkdir` — создаёт папку

---

## Дополнения

**`cp -r папка`** — копировать папку рекурсивно
**`cp -i`** — спросить перед перезаписью

**`mv файл папка`** — переместить файл в папку
**`mv старое новое`** — переименовать

**`rm -r папка`** — удалить папку рекурсивно
**`rm -f`** — удалить без подтверждения
**`rm -rf`** — удалить всё без вопросов ⚠️

**`mkdir -p папка/папка`** — создать вложенные папки

---

## Примеры

**`cp file.txt backup.txt`** → скопировал файл
**`cp -r Documents/ Backup/`** → скопировал папку

**`mv file.txt Documents/`** → переместил файл
**`mv old.txt new.txt`** → переименовал файл

**`rm file.txt`** → удалил файл
**`rm -rf папка/`** → удалил папку со всем содержимым

**`touch file.txt`** → создал пустой файл

**`mkdir projects`** → создал папку
**`mkdir -p projects/linux/scripts`** → создал вложенные папки
