# Глава 2. Операции с файлами

> [!info] Зачем это знать
> Создавать, копировать, перемещать, удалять файлы — базовые операции которые используешь каждый день.

---

## Просмотр файлов

### cat file.txt
```
> показывает содержимое файла целиком
```

### less file.txt
```
> открывает файл для чтения с прокруткой (q — выйти)
```

### head file.txt
```
> показывает первые 10 строк файла
```

### head -n 20 file.txt
```
> показывает первые 20 строк файла
```

### tail file.txt
```
> показывает последние 10 строк файла
```

### tail -f file.txt
```
> следит за файлом в реальном времени — новые строки появляются сразу
```

---

## Создание файлов и папок

### touch file.txt
```
> создать пустой файл
```

### mkdir folder
```
> создать папку
```

### mkdir -p folder/subfolder/subsubfolder
```
> создать вложенные папки сразу
```

### nano file.txt
```
> открыть файл в редакторе nano (Ctrl+X — выйти, Ctrl+S — сохранить)
```

---

## Копирование

### cp file.txt copy.txt
```
> скопировать файл
```

### cp file.txt /path/to/dir/
```
> скопировать файл в другую директорию
```

### cp -r folder/ /path/to/dir/
```
> скопировать папку со всем содержимым
```

---

## Перемещение и переименование

### mv file.txt /path/to/dir/
```
> переместить файл в другую директорию
```

### mv oldname.txt newname.txt
```
> переименовать файл
```

### mv folder/ /path/to/dir/
```
> переместить папку
```

---

## Удаление

### rm file.txt
```
> удалить файл
```

### rm -r folder/
```
> удалить папку со всем содержимым
```

### rm -rf folder/
```
> удалить папку принудительно без подтверждения
```

> [!warning] Осторожно
> `rm -rf` удаляет безвозвратно. Нет корзины — файлы не восстановить. Особенно осторожно с `rm -rf /`

---

## Поиск файлов

### find / -name file.txt
```
> найти файл по имени во всей системе
```

### find /home -name "*.txt"
```
> найти все .txt файлы в /home
```

### find / -name file.txt -type f
```
> найти только файлы (не папки)
```

### find / -name folder -type d
```
> найти только папки
```

---

## Поиск внутри файлов

### grep "текст" file.txt
```
> найти строки содержащие текст в файле
```

### grep -r "текст" /path/
```
> найти текст во всех файлах в директории
```

### grep -i "текст" file.txt
```
> найти текст без учёта регистра
```

### grep -n "текст" file.txt
```
> найти текст и показать номера строк
```

---

## Информация о файлах

### file file.txt
```
> показывает тип файла
```

### wc -l file.txt
```
> считает количество строк в файле
```

### du -sh folder/
```
> показывает размер папки
```

### df -h
```
> показывает свободное место на дисках
```

## Ссылки

### ln -s /path/to/file link
```
> создать символическую ссылку (как ярлык в Windows)
```
