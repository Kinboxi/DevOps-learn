# Навигация

# pwd — показывает где ты находишься
# ls — показывает что находится в папке
# cd — переходит в другую папку

---

## Дополнения

**`ls -l`** — подробный список
**`ls -a`** — скрытые файлы
**`ls -la`** — подробно + скрытые

**`cd ..`** — уровень выше
**`cd ~`** — домашняя папка
**`cd /`** — корень системы
**`cd -`** — прошлая папка

---

## Примеры

**`pwd`** → `/home/user`

**`ls`** → `file.txt folder video.mp4`
**`ls -a`** → `.bashrc .ssh file.txt`

**`cd Documents`** → `/home/user/Documents`
**`cd ..`** → `/home/user/Documents` → `/home/user`
