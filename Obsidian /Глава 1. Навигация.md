## Структура файловой системы

В Linux всё начинается с корня — `/`. Это как диск C: в Windows но только один на всю систему.

```
/
├── home/     — домашние папки пользователей
├── etc/      — конфиги программ
├── var/      — логи, базы данных, переменные данные
├── usr/      — программы и утилиты
├── tmp/      — временные файлы
├── root/     — домашняя папка root
└── bin/      — основные команды системы
```

> [!tip] Аналогия
> `/` это как подъезд дома. Все квартиры (папки) находятся внутри.

## Перемещение

### cd /path/to/dir
```
> перейти в указанную директорию
```

### cd ~
```
> перейти в домашнюю директорию
```

### cd ..
```
> перейти на уровень выше
```

### cd -
```
> вернуться в предыдущую директорию
```

### cd /
```
> перейти в корень файловой системы
```

## Просмотр содержимого

### ls
```
> показывает файлы и папки в текущей директории
```

### ls -l
```
> подробный список — права, владелец, размер, дата
```

### ls -a
```
> показывает скрытые файлы (начинаются с точки)
```

### ls -la
```
> подробный список включая скрытые файлы
```

### ls -lh
```
> подробный список с читаемым размером файлов (KB, MB, GB)
```

### ls /etc
```
> посмотреть содержимое любой директории не заходя в неё
```

## Абсолютный и относительный путь

**Абсолютный** — начинается с `/`, всегда от корня:
```
/home/user/documents/file.txt
```

**Относительный** — от текущей директории:
```
documents/file.txt
./documents/file.txt
```

> [!tip]
> `.` — текущая директория
> `..` — родительская директория

---

## Полезные символы

| Символ | Значение |
|--------|----------|
| `~` | домашняя директория |
| `.` | текущая директория |
| `..` | родительская директория |
| `/` | корень файловой системы |
| `-` | предыдущая директория |
