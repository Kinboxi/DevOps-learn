# Права и доступ

# `chmod` — изменить права на файл
*`chmod 755 file.txt` → владелец всё, остальные читать и запускать*

# `chown` — изменить владельца файла
*`chown ivan file.txt` → владелец теперь иван*

# `chgrp` — изменить группу файла
*`chgrp devs file.txt` → группа теперь devs*

# `ls -l` — посмотреть права
*`ls -l` → `-rwxr-xr-- ivan devs file.txt`*

---

## Как читать права

# `-rwxr-xr--`
*`-` — файл / `d` — папка / `l` — ссылка*
*`rwx` — владелец*
*`r-x` — группа*
*`r--` — остальные*

**`r`** — читать `4`
**`w`** — писать `2`
**`x`** — выполнять `1`

---

## Биты

| Число | Права | Что значит |
|-------|-------|------------|
| `7` | `rwx` | всё |
| `6` | `rw-` | читать и писать |
| `5` | `r-x` | читать и запускать |
| `4` | `r--` | только читать |
| `0` | `---` | ничего |

*`755` → владелец всё, группа читать/запускать, остальные читать/запускать*
*`644` → владелец читать/писать, группа читать, остальные читать*
*`777` → все могут всё ⚠️*

---

## Специальные биты

# `SUID` — запускает файл от имени владельца
*`chmod u+s file`*
*`chmod 4755 file`*
*`ls -l` → `-rwsr-xr-x` — `s` вместо `x` у владельца*

# `SGID` — запускает от имени группы
*`chmod g+s file`*
*`chmod 2755 file`*
*`ls -l` → `-rwxr-sr-x` — `s` вместо `x` у группы*

# `Sticky bit` — только владелец может удалить свой файл
*`chmod +t папка/`*
*`chmod 1755 папка/`*
*`ls -l` → `drwxr-xr-t` — `t` в конце*

| Бит | Число | Где используется |
|-----|-------|-----------------|
| `SUID` | `4` | исполняемые файлы |
| `SGID` | `2` | папки и файлы |
| `Sticky` | `1` | папки `/tmp` |

*`/tmp` имеет sticky bit — каждый создаёт файлы но удалить может только свой*

## Примеры

# `chmod`
*`chmod 755 file.txt` → стандартные права*
*`chmod 644 file.txt` → только владелец пишет*
*`chmod +x file.sh` → добавить запуск*
*`chmod -x file.sh` → убрать запуск*
*`chmod -R 755 папка/` → применить ко всей папке*

# `chown`
*`chown ivan file.txt` → сменить владельца*
*`chown ivan:devs file.txt` → сменить владельца и группу*
*`chown -R ivan папка/` → применить ко всей папке*
