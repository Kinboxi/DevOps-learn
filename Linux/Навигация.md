## 📂 Команды навигации (Терминал)
Эти команды и символы помогают перемещаться по папкам:

| Команда / Символ | Что делает |
| :--- | :--- |
| **`pwd`** | Показывает полный путь к папке, где вы сейчас (Print Working Directory). |
| **`ls`** | Выводит список всех файлов и папок в текущей директории. |
| **`cd`** | Переход в указанную папку (например, `cd Documents`). |
| **`~`** | Тильда — быстрый путь в вашу **домашнюю папку** (`/home/user`). |
| **`/`** | Слэш — **корень** всей системы (самый верхний уровень). |
| **`..`** | Две точки — шаг на один уровень **назад** (в родительскую папку). |
| **`.`** | Одна точка — обозначает **текущую** папку. |

---

### Примеры использования:
* `cd ..` — выйти из папки на уровень выше.
* `cd ~/Pictures` — быстро перейти в папку с картинками из любого места.
* `ls -a` — показать вообще все файлы, включая скрытые.
