# Linux Navigation (Simple English)

## 📁 Basic Commands
* **`pwd`** — **Where am I?** (Shows your current folder).
* **`ls`** — **Show files.** (Lists all files and folders here).
* **`cd [folder]`** — **Go to folder.** (Moves you inside a new place).

---

## 📍 Special Symbols (Shortcuts)
These symbols help you move faster:

1.  **`~`** (Tilde) — **Home.** Your main personal folder.
2.  **`/`** (Slash) — **The Root.** The very top of the system.
3.  **`..`** (Two dots) — **Go Back.** Moves you one level up.
4.  **`.`** (One dot) — **Here.** The current folder you are in.

---

## 💡 Examples
* `cd ..` — Move one folder back.
* `cd ~` — Go to your Home folder quickly.
* `ls ..` — See files in the folder above you.
